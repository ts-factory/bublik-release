Release v1.2.0

February 18, 2025 · 2 min read
Kostromin Danil
Bublik UI Maintainer
Rybchenko Natalia
Bublik Backend Maintainer

🎉 Exciting Updates in the Latest Release! 🎉

This release introduces key improvements, including enhanced report visualization, bug fixes, and optimizations for a smoother experience.

🔍 What's New?​

🚀 New Feature​

  • Report Charts: Failed results are now highlighted in red with a diamond symbol, making it easier to identify issues at a glance.

🐛 Bug Fixes​

  • Run Table: Fixed an issue where column visibility was not properly retained in the URL state.

📦 Chores & Maintenance​

  • Removed a redundant config migration check to streamline configuration handling.

These updates enhance usability, improve reliability, and ensure a more seamless experience. 🚀

Backend Update​

  1. cd bublik
  2. git remote update
  3. git checkout v1.2.0
  4. ./scripts/deploy --steps pip_requirements django_settings run_services

Frontend Update​

  1. Trigger the workflow in your frontend repository.
  2. Synchronize the mirrors.
  3. cd bublik-ui
  4. git remote update
  5. git checkout v1.2.0
